# **App Name**: BolsoZen

## Core Features:

- Expense Tracking: Log monthly fixed and variable expenses with categorization.
- Credit Card Management: Record and manage credit card expenses.
- Installment Calculation: Automatically calculate future installments for tracked expenses.
- Income Recording: Log salaries, earnings, and extra income, differentiating between recurring and occasional income.
- Spending Analysis Tool: Analyze financial data using generative AI to provide personalized insights, offering users advice based on identified trends.
- Visual Reports: Provide charts and dashboards to track income, expenses, and monthly balance.
- Comparative Analysis: Allow users to compare financial performance across previous months.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke sophistication and financial acumen.
- Background color: Very light purple (#F3E5F5).
- Accent color: Electric blue (#3F51B5) to highlight interactive elements and important data.
- Body and headline font: 'PT Sans' for a modern yet approachable feel, suitable for all text.
- Use clean, minimalist icons to represent different expense and income categories.
- Mobile-first design with a clear, intuitive layout for easy navigation.
- Subtle animations for transactions and data updates to enhance user experience.
